# Factorial Recursion Output

```
Enter a number: 5
Factorial of 5 is 120
```

## Code Explanation

This program calculates the factorial of a given number using recursion. For input 5, it calculates 5! = 5 × 4 × 3 × 2 × 1 = 120.
